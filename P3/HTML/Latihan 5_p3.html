Belajar CSS Box Model

apa itu box model

Setiap elemen HTML dianggap sebagai kotak persegi panjang oleh browser. Box model terdiri dari:

  • Content: Isi sebenarnya dari element
  • Padding: Ruang antara content dan border
  • Border: Garis tiap element
  • Margin: Ruang antara border dan elemen lain
